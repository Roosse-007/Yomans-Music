# Yomans Music

Web service streaming musik berbasis REST API.
## Deskripsi
Project ini merupakan pengembangan web musik yang mendukung:
- Streaming audio
- Manajemen lagu
- Upload audio
- Autentikasi user (JWT)
- Backend RESTful API
Dikembangkan menggunakan Laravel.
